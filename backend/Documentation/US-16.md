# US-16: Navigation Route API

## Overview
This feature implements an endpoint to retrieve navigation route information for a specific destination. This allows users to plan their travel by viewing start points, waypoints, end points, estimated travel time, and distance.

## Endpoint
**GET** `/api/navigation/:destinationId/`

### URL Parameters
- `destinationId`: The ID of the destination for which navigation information is requested.

## Response
### Success (200 OK)
Returns a JSON object containing the navigation details:
```json
{
    "startPoint": "Kathmandu",
    "waypoints": ["Nagarkot", "Kurintar"],
    "endPoint": "Pokhara",
    "estimatedTimeMinutes": 360,
    "distanceKm": 200.5
}
```

### Errors
- **404 Not Found**: Returned if the destination does not exist or if no navigation route is defined for the specified destination.

## Implementation Details
1.  **Model**: A new `Navigation` model was created in `api/models/navigation.py`. It has a one-to-one relationship with the `Destination` model.
    - `destination`: OneToOneField(Destination)
    - `start_point`: CharField
    - `waypoints`: JSONField (to store a list of points)
    - `end_point`: CharField
    - `estimated_time_minutes`: IntegerField
    - `distance_km`: FloatField
2.  **Serializer**: `NavigationSerializer` in `api/serializers/navigation.py` handles the mapping between snake_case database fields and camelCase API response fields.
3.  **View**: `NavigationView` in `api/views/navigation.py` fetches the destination and its associated navigation record, returning 404 if either is missing.
4.  **Tests**: Comprehensive tests were added in `api/tests_navigation.py` to verify success and error cases.

## How to Test in Postman
1.  **Seed Data**: Ensure you have a destination and an associated navigation record in the database.
2.  **Request**: Send a GET request to `http://127.0.0.1:8000/api/navigation/<id>/` where `<id>` is an existing destination ID.
3.  **Verify**: Check that the response contains the expected route details with camelCase keys.
